# v0.8.0 — Admin enrichment + auth fix (Phase I.5)

Pairs with simulator **v0.14.0**. Adds expanded admin management UI and
fixes a token timing edge case during login.

## New features

### Admin UI

- Enriched admin screens for user management and wallet controls.
- Wallet freeze/unfreeze actions surfaced in the admin panel.
- Mempool display remains read-only while admin operations update state.

## Fixes

- `GET /auth/me` is no longer called before the JWT is available after
  login, preventing spurious 401s.
