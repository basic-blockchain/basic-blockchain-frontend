## 1. Goal & non-goals

### Goal

Third long-tail sub-phase of v2. Migrate the configuration views to
consume Phase 7.1 atoms without changing data sources or behavior.

### In scope

- `src/views/AdminSettingsView.vue`
- `src/views/AdminCurrenciesView.vue`
- `src/views/AdminExchangeRatesView.vue`

Shared migration rules (all three views):

- Bigstat KPIs -> `<BaseCard variant="bigstat">`
- Inline tables -> `<BaseTable>` + `cell-<key>` slots
- Status/type pills -> `<BaseBadge>`
- Inline buttons -> `<BaseButton>`
- Inline panels -> `<BaseCard>` (default, padding as needed)
- Drop scoped CSS for `.bigstat*`, `.tbl*`, `.btn*`, `.panel*`,
  `.status-dot*`, `.bdg*` once absorbed by atoms.

### Non-goals

- No backend/API changes.
- No router changes.
- No new dependency.
- No AssetPill extraction (still only two consumers).
- No BaseInput/BaseSelect extraction (keep native controls).

## 4. Props / structure changes

### 4.1 AdminSettingsView

- Replace `.panel` sections with `<BaseCard>`:
  - General panel: `BaseCard` with header/footer slots to host the
    title/description and Save button.
  - Toggles panel: `BaseCard` with internal list rows.
- Replace sidebar buttons with BaseButton `variant="ghost"` +
  `size="sm"` to normalize chrome.
- Keep native inputs/selects and toggles inline (no BaseInput).

### 4.2 AdminCurrenciesView -> BaseTable columns

```ts
const columns = [
  { key: 'asset', label: 'Activo' },
  { key: 'code', label: 'Codigo' },
  { key: 'name', label: 'Nombre' },
  { key: 'decimals', label: 'Decimales', num: true },
  { key: 'type', label: 'Tipo' },
  { key: 'status', label: 'Estado' },
]
```

Slot map:

- `cell-asset`: keep `.asset-pill` (no extraction yet).
- `cell-type`: `BaseBadge` tones:
  - stablecoin -> `success`
  - platform -> `warning`
  - native -> `accent` or `info`
- `cell-status`: `BaseBadge` tones:
  - active -> `success`
  - inactive -> `neutral`

### 4.3 AdminExchangeRatesView -> BaseTable columns

```ts
const columns = [
  { key: 'pair', label: 'Par' },
  { key: 'rate', label: 'Tasa' },
  { key: 'fee', label: 'Comision' },
  { key: 'source', label: 'Fuente' },
  { key: 'updated', label: 'Actualizado' },
]
```

Slot map:

- `cell-pair`: mono pair string.
- `cell-rate` / `cell-fee`: mono numeric.
- `cell-source`: `BaseBadge` tones:
  - MANUAL -> `accent` (or `info`)
  - BINANCE/CRYPTO_COM -> `success`
- `cell-updated`: mono timestamp.
